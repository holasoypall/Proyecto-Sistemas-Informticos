MBR: Registro de arranque principal

El MBR es el primer sector físico de un dispositivo de almacenamiento de datos que arranca el ordenador, esto lo hace a través de la BIOS.
En los dispositivos de almacenamiento que contienen los datos para el proceso de arranque, a las particiones se les asigna un máximo de 2TB, esto se debe a las limitaciones del momento. Su dirección específica en el sector del disco es Cilindro 0, Cabeza 0, Sector 1.
Para el proceso de arranque, MBR consta de tres partes: el código de arranque maestro, la tabla de partición maestra, y la firma del disco.

Las tareas de la MBR se pueden dividir en 3 partes:

  • La BIOS busca un dispositivo de destino para arrancar el MBR cuando un ordenador se enciende por primera vez.

  • El código de arranque MBR lee la tabla de particiones e identifica la partición “activa”.

  • Utiliza el sector de arranque de esa partición para iniciar el sistema operativo.

MBR vs GPT

GPT (Tabla de partición GUID) es la forma de tablas de particiones moderna, que acaba con algunas de las limitaciones de MBR.

  • MBR

    • Máximo de 4 particiones primarias, tamaño máximo de 2TB por disco.
    • Compatible con sistemas operativos antiguos y BIOS.
    • Adecuado para discos de como máximo 2TB y sistemas sin UEFI.

  • GPT

    • Admite un número ilimitado de particiones, sin límite de 2TB.
    • Optimizado para sistemas UEFI.
    • Adecuado para discos de más de 2 TB y para mayor capacidad y flexibilidad.
