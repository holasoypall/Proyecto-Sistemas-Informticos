Turning on a computer is only the start of a perfect secuence between hardware and software. Each component takes part silently until the operating system takes control and then allows the user to use it.

Introduction to the Boot System

Far before the operating system appears on the screen, the computer tries to follow a series of important steps that allow everything to work properly. These are the fundamentals components of the process:

  • ROM Memory

    Contains the initial instructions that allow the computer to start and begin the boot process

  • BIOS

    Supervises the hardware, runs test and identifies from which device the operating system will be loaded

  • MBR / EFI

    Manage the loading of the operative system, and ensure that it properly passes to the software
