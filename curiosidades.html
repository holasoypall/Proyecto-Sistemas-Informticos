Curiosidades


Existen varios tipos de arranque:

  • Arranque desde disco duro: el más común.

  • Arranque desde USB o CD/DVD: para instalar o reparar sistemas.

  • Arranque en red (PXE): usado en empresas para cargar sistemas desde un servidor.

  • Arranque dual/múltiple: tener varios sistemas operativos (Windows + Linux, por ejemplo).

  • Arranque seguro (Secure Boot): solo permite cargar sistemas firmados digitalmente (evita malware en el arranque).


Más curiosidades:

  • En los Mac, el equivalente a la BIOS es el EFI Apple Boot Manager.

  • El tiempo de arranque puede variar mucho según el firmware, el tipo de disco (HDD vs SSD) y si está activado el Fast Boot.

  • Algunos ordenadores antiguos necesitaban discos de arranque (como el famoso “A:>” de MS-DOS).


¿Como funciona el sistema de arranque?

Cuando enciendes un ordenador este no arranca directamente el sistema operativo, antes ocurren una serie de pasos muy controlados para comprobar el hardware y cargar el software básico que permitirá iniciar el sistema.


Etapas principales del sistema de arranque:

  • La corriente eléctrica llega a la placa base.

  • Ejecución del firmware BIOS/UEFI: el código grabado en la memoria ROM se ejecuta.

  • POST o Power-On Self Test: se comprueban memoria RAM, CPU, teclado, etc.

  • Búsqueda del dispositivo de arranque como disco, USB, red, etc.

  • Carga del gestor de arranque o bootloader ubicado en el MBR o en la partición EFI.

  • Finalmente se inicia el sistema operativo.
