Curiosities


There are several types of startup:

  • Booting from hard drive: the most common.

  • Boot from USB or CD/DVD: to install or repair systems.

  • Network boot (PXE): used in companies to load systems from a server.

  • Dual/multiple boot: having several operating systems (Windows + Linux, for example).

  • Secure Boot: only allows loading digitally signed systems (prevents malware at startup).


More curiosities:

  • On Macs, the equivalent of the BIOS is the EFI Apple Boot Manager.

  • Boot time can vary greatly depending on the firmware, the type of disk (HDD vs SSD), and whether Fast Boot is enabled.

  • Some older computers required boot disks (like the famous MS-DOS "A:>" disk).


How does the boot system work?

When you turn on a computer, it doesn't directly boot into the operating system. First, a series of highly controlled steps occur to check the hardware and load the basic software that will allow the system to start.


Main stages of the starting system:

  • The electrical current reaches the motherboard.

  • BIOS/UEFI firmware execution: the code recorded in the ROM memory is executed.

  • POST or Power-On Self Test: checks RAM, CPU, keyboard, etc.

  • Search for boot device such as disk, USB, network, etc.

  • Loading the boot manager or bootloader located in the MBR or in the EFI partition.

  • Finally, the operating system starts.
