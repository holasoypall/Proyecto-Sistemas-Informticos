ROM: Read-Only Memory

ROM stands for Read-Only Memory. It is a type of storage characterized by being non-volatile. This means that the information stored in it is not erased when you turn off the device; the data remains stored.
The essential function of ROM is to store the firmware or boot software (such as the BIOS or UEFI in a computer). This is the set of instructions that tells the hardware how to start the operating system when you turn on the computer.
We can think of ROM as the computer's permanent instruction manual.

History of the ROM

  • 1950-60: Headed Cores (Manual)
  • 1970 (beginnings): MROM (Factory)
  • 1970 (finals): EPROM (UV erasable)
  • 1980-90: EEPROM (Electrically erasable)
  • Actualidad: Flash Memory (Fast and flexible)

IMPORTANT ! ! !

It is essential not to confuse ROM with RAM (Random-Access Memory).

  • ROM is the "permanent brain": Stores instructions that never change.
  • RAM is the "temporary table": It stores programs and files you're using in real time. It's volatile (erased when you turn off) and is both readable and writable.

When you turn on your computer, the CPU (the processor) is designed to do one thing: find an instruction at a specific memory address. This address points directly to the ROM chip.
The ROM takes control and sends the POST (Power-On Self-Test) to the CPU, a mini-program that checks the basic hardware to ensure it is present and functioning. If everything is okay, the ROM hands control over to the next step, which is the BIOS or UEFI.

Although we're talking about PC booting here, ROM is in almost every device: a microwave, a car, a printer...

Common Types of ROM

  • MROM (Mask ROM):
    The "pure" ROM. The data is written at the factory during chip manufacturing. It is impossible to modify.
  • RPOM (Programmable ROM):
    It is manufactured "in blank" and can be programmed only once by a user with a special device.
  • EPROM (Erasable Programmable ROM):
    It can be erased by exposing it to strong ultraviolet light and then reprogramming it.
  • EEPROM (Electrically Erasable Prog. ROM):
    The most modern one. It can be erased and reprogrammed electrically, without removing it from the circuit.
