DEL ENCENDIDO
AL SISTEMA OPERATIVO
DEL ORDENADOR

Encender un ordenador es solo el comienzo de una coreografía perfecta entre hardware y software. Cada componente participa en silencio hasta que el sistema operativo toma el control.

Introducción al Sistema de Arranque

Antes de que el sistema operativo aparezca en pantalla, el ordenador realiza una serie de pasos esenciales que permiten que todo funcione correctamente. Estos son los protagonistas del proceso

  • Memoria ROM

    Contiene las instrucciones iniciales que permiten que el ordenador despierte y comience el arranque

  • BIOS

    Supervisa el hardware, realiza pruebas y determina desde qué dispositivo se cargará el sistema

  • MBR / EFI

    Gestionan la carga del sistema operativo, asegurando que el control pase correctamente al software.
